loadMeThere()

Ajax toolset - something that should be built in!

GitHub
What is it doing?

In simple words: it loads a requested URL with Ajax and puts the loaded content (answer) into a target.

As a very basic example: you have a DIV container like this one:
<div id="mySpecialDIV">Some predefined content in my special DIV</div>

And you have a basic link like that:
<a href="https://mydomain.com/some/file.php?foo=bar">My Link</a>

Only with adding some parameters to the link above, we are able to load the href-URL into the DIV with the ID mySpecialDIV.
In fact we can say a bit more than that :)
The cool part of loadMeThere is that is not only a function that you might use in a JS script, it's real potential comes with the listeners that will trigger the magic of Ajax. With the listeners we can say that any element that has a class "loadmethere" can be used to trigger the loadMeThere() function and get content from a URL.

JS: How to (minimum)

I won't go to deep in this part. If you're developing in JS, the following infos with the "HTML: How to"-knowledge will be enough. If not: tell me what you're missing.

  • Be sure to use (preloaded) jQuery in your project.
  • Of course you will need to include loadmethere.js in your project too!
  • When ever you fire the function loadMeThere() it will load the URL into the target:
    loadMeThere(requestURL, targetContainer, position, scrollToTarget, disableCustomElement, disableAnyLMT, animationTime, scrollToTargetOffset);

HTML: How to (minimum)
  • Be sure to add jQuery first! CDN example:
    <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js" integrity="sha384-DfXdz2htPH0lsSSs5nCTpuj/zy4C+OGpamoFVy38MVBnE+IbbVYUew+OrCXaRkfj" crossorigin="anonymous"></script>

  • First you need to include loadmethere.js or loadmethere.min.js in your HTML (can be in the footer):
    <script src="path/to/loadmethere.min.js"></script>

  • Next we need the target. Where should the requestet URLs anser appear? This can be any element as long as it has a id tag (id="").
    <div id="mySpecialDIV">
      this container can already be filled or be empty
    </div>

  • Magic comes now: The element that triggers the Ajax call.
    This can also be any element as long as it has the class "loadmethere" and some data-tags set. To keep it simple and a bit better to understand we're using a basic link:
    <a href="https://mydomain.com/file.php" class="loadmethere" data-target="mySpecialDIV" data-position="in">My Link</a>
  • That's all! If you click on the link now, the URL should be loaded and printed in the above DIV container.
Browser URL History

loadMeThere comes with a simple but effective way to handle the browsers URL history! The trick is the use of the default href tag and a custom data-href tag. Depending on which of them you use in your loadMeThere link, The history will be pushed or not. You can choose between three ways:

  1. href and data-href
    This will load the data-href URL with Ajax and push href to browser URL history.

  2. href only
    This will load the href URL with Ajax and also push href to browser URL history.

  3. data-href only
    This will load the data-href URL with Ajax and push wont push anything.

Examples

Every example has the following versions / differences:

  1. HREF ONLY
    On click it will load the file at the desired location and push the same URL to browsers hiostory. Check your URL bar (card-xx.html). If you refresh the site you will load the Ajax file only if your code won't catch this in any way.
  2. DATA-HREF ONLY
    On click it loads the URL given in data-href with Ajax - no URL push to browser. On a refresh you will load this page again, because nothing has changed for the browser. Usefull if you want to trigger something but don't want to load it again on a page refresh.
  3. HREF & DATA-HREF
    This will load data-href URL with Ajax but push the href URL into browsers top bar.
    On page refresh the pushed href URL will be reloaded. You can use this if you want to pass a GET var to the URL (maybe a filename). Later in your code you can check if the GET var is set and maybe load the file again or what ever.

Scroll to Target

To see effect first load multiple examples of a prepend link. The button will load a HTML file before the target and scroll to it.

#1 Scroll Offset 200 #2 Scroll Offset -200

Same like "Scroll to Target" link, first with 200px offset and second with -200px. Usefull if you have a navbar where the target might be at the wrong position after scrolling to it.

Disable all LMT

This will only work on a web server with PHP! It will load the file "card-slow.php" with position "in", the PHP file has a sleep(3) to simulate longer loading time so you can see the "data-disable-lmt" effect where all loadMeThere links will be disabled while loading.

You can combine this with "data-disable-element".

Disable Element

This will only work on a web server with PHP! It will load the file "card-slow.php" with position "in", the PHP file has a sleep(3) to simulate longer loading time so you can see the "data-disable-element" effect where a custom element ("#id") or elements (".class") will be disabled while loading. In this case we are disableing the DIV with id="disable-example" (which is this hole row).

You can combine this with "data-disable-lmt".

Note!
  • In some cases you need to re-init the newly loaded stuff (especially for jQuery listeners or functions).
  • I also noticed (sometimes) that if your answer contains a <script>...</script> part, it will print it in the browser instead of executing it. Workaround: just put the script part into a DIV (wrap it).
Additional Functions

This script comes with two more functions and the according listeners for them. Actually they are not in the examples but here is a short how to for now:

  • emptyTarget(targetContainer, animationTime)
    This function will empty the content of a target element (ID). The element itself will remain, only it's content gets removed.
    Example:
    <a href="#" class="emptytarget" data-target="mySpecialDIV" data-animation-time="350">Empty the Target</a>

  • deleteTarget(targetContainer, animationTime)
    This function will remove the target element (ID) completely.
    Example:
    <a href="#" class="deletetarget" data-target="mySpecialDIV" data-animation-time="350">Empty the Target</a>
